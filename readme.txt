LUCKY GAME
В этой игре нужно угадать число, если вы угадаете число то вас просто поздравят с выиграшем, а в случае поражения у вас удалится система на компьютере. Открывается только от именни суперпользователя/root, если у вас не запускается то установите модули такие как: tk, python.
Для операционных систем на базе ядра linux открывайте файл main.py
Для windows открывайте win.py
___

In this game you need to guess the number, if you guess the number then you will simply be congratulated on winning, and in case of defeat, the system on your computer will be deleted. It opens only on behalf of the superuser/root, if you do not start, then install modules such as: tk, python.
For operating systems based on the linux kernel, open the main.py file
For windows, open win.py
